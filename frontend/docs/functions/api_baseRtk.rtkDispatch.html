      Asynchronously dispatches an RTK Query action using the provided Redux store, waits for the query result, and then unsubscribes from the query to clean up resources.

      This utility handles the retrieval of query data and ensures proper resource cleanup by unsubscribing from the query after execution. The function is generic, allowing you to type the expected return data of the query.
