Function addRole

Asynchronously adds a role to a specified user.

This function sends a PATCH request to update the role assigned to a user. It constructs the request using the user's ID and the role's ID. The operation is abortable using an AbortSignal provided via the signal parameter.

Throws

Throws an error if the fetch operation fails or if the response indicates a failure.
