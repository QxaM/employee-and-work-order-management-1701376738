    • useMeData(): { isError: boolean; isLoading: boolean; me: undefined | MeType }

      useMeData is a custom hook that manages the state and behavior related to user authentication and data retrieval.

      useMeData

      Returns { isError: boolean; isLoading: boolean; me: undefined | MeType }

      An object containing the following:

      • me (Object|null): The user data retrieved from the useMeQuery hook. Returns null if the data is not available.
      • isLoading (boolean): Indicates whether the data is currently being loaded.
      • isError (boolean): Indicates whether there was an error during the data retrieval process.

      Description

      This hook uses useMeQuery to fetch the user data. It monitors the authentication token and error state to ensure proper logout behavior. If an error occurs while a valid token is set, the logout action is dispatched to clear the user's session. The logout operation is managed using a useRef to prevent redundant dispatches.
